### File Organiser 🗂️

Watches my Downloads folder and quietly sorts chaos into folders.

```
Downloads/
├── Images/      .jpg  .png  .gif
├── PDFs/        .pdf
├── Videos/      .mp4  .mov
├── Music/       .mp3  .wav
└── Code/        .py   .js   .html
```

`os` · `shutil` · `pathlib`

### My Email Bot 📬

Sends reminder emails on a schedule — morning, afternoon, night.

- 🌅 8 AM — morning nudge
- 📚 2 PM — study check-in
- 🌙 10 PM — wind down

> Set up `config.py` with your Gmail credentials before running.

`smtplib` · `schedule`
